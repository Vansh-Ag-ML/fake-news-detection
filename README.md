# 📰 Fake News Detection System

## 📌 Overview
This project is a Machine Learning-based system that classifies news articles as **Fake** or **Real** using Natural Language Processing (NLP) techniques.

---

## 🧠 Problem Statement
With the rapid spread of misinformation online, it becomes essential to automatically identify fake news. This project aims to build a reliable model that can analyze textual content and predict its authenticity.

---

## ⚙️ How It Works
- Text data is preprocessed and cleaned
- TF-IDF Vectorization converts text into numerical features
- A Machine Learning model predicts whether the news is Fake or Real

---

## 🧰 Tech Stack
- Python  
- Pandas  
- Scikit-learn  
- TF-IDF Vectorizer  

---

## 🤖 Model Details
- Algorithm: Logistic Regression / Passive Aggressive Classifier  
- Feature Extraction: TF-IDF  
- Evaluation Metric: Accuracy  

---

## 📈 Results
- Achieves good accuracy on test data  
- Performs well on detailed news content  
- Slight limitations with short headlines  

---

## ▶️ How to Run
1. Open the notebook in Google Colab or VS Code  
2. Install required libraries (`pandas`, `sklearn`)  
3. Run all cells  
4. Enter news text to get prediction
